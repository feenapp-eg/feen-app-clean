# Feen (Vite + React)

## Deploy on Vercel
1. Push this repo to GitHub.
2. Import to Vercel.
3. In Vercel → Settings → Environment Variables, add:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
   - VITE_PROFILE_URL (optional)
   - VITE_USE_SUPABASE (optional)

## Local dev (optional)
npm install
npm run dev
